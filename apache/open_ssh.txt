Pour héberger mon site et le rendre accessible sur internet, j'utilise apache, une fois chargé avec :

- apt-get install apache2

Dans le répertoire "etc" on retrouvera le dossier "apache2", on rentre dedans puis dans "sites-available".

On copie le fichier "000-default.conf" en le modifiant avec le nom de mon site :

- cp 000-default.conf monsite.local.conf

A l'intérieur de ce fichier je paramètre :

ServerName monsitetest.local
ServerAlias www.monsitetest.local
DocumentRoot /home/monsitetest/www	** le lieu où j'ai mis mon site, ça pourrait être dans var/www"**

Je redémarre mon serveur :cd cnan

- sudo systemctl restart apache2.service

J'active mon site avec la commande :

- sudo a2ensite monsitetest.local

/////////////////////////////////////////////////////

Si ne fonctionne pas, allez dans etc/apache2/apache2.conf ajouter :

<Directory /home/>
	Options Indexes FollowSymLinks
	AllowOverride all
	Require all granted
</Directory>

/////////////////////////////////////////////////////

Pour tester en local sur ma machine windows, je cherche l'application bloc-notes, je fais un click droit pour ouvrir en tant qu'administrateur, puis je click sur fichier, je me rends dans "C:\Windows\System32\drivers\etc" je fais "afficher tout", puis je sélectionne le fichier "hosts" je rentre l'ip de ma vm puis l'adresse de mon site :

- 192.168.1.97 monsitetest.local
